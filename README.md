# 信用卡客户违约风险分析与预测

本项目基于 UCI `Default of Credit Card Clients` 数据集，分析客户历史账单、还款和逾期行为与下一期违约之间的关系。第一阶段聚焦数据理解、质量检查、探索性分析、风险指标构造和规则式客户风险分层，暂不训练机器学习模型。

## 项目目的

- 用可复现的数据流程回答信用风险业务问题。
- 展示 Pandas、NumPy、Matplotlib 和统计分析能力。
- 为后续机器学习、阈值与误判成本分析建立可靠基础。

## 第一阶段结果

- 数据包含 **30,000** 条记录和 **24** 个字段，目标变量无缺失；发现 **35** 条完全重复记录。由于缺少可靠的客户唯一标识，本阶段只报告、不武断删除。
- 样本总体违约率为 **22.12%**。
- 没有历史逾期的客户违约率为 **11.71%**；六个月均出现逾期的客户违约率为 **70.32%**，历史逾期频次与后续违约呈明显正相关。
- 规则式低、中、高风险群体的违约率分别为 **11.71%**、**32.51%** 和 **59.99%**。该分层用于探索性描述，不是真实授信模型。
- `<=100k` 授信额度组违约率为 **29.48%**，`>500k` 组为 **11.17%**；这可能同时受到客户筛选与既有风控策略影响，不能解释为提高额度会降低违约。
- 年龄组违约率呈非单调差异：30–39 岁组为 **20.25%**，60 岁及以上组为 **28.32%**。60+ 组只有 339 人，解读时需注意样本量。
- 额度使用率从 `0–25%` 上升到 `75–100%` 时，违约率从 **16.65%** 上升到 **30.74%**；使用率超过 100% 的 637 名客户违约率为 **34.22%**。
- 还款比例为 `0–10%` 的客户违约率为 **26.99%**，高于还款比例超过 100% 客户的 **14.47%**；另有 966 名客户因正账单总额为零而无法定义该比例。
- 平均还款金额最低四分位组违约率为 **31.16%**，最高四分位组为 **13.23%**；平均账单金额四分位的违约率差异较弱，说明金额本身不如还款行为稳定，但这些关系仍可能受授信额度等因素影响。
- 对同样逾期 2 个月的客户，连续逾期组违约率为 **37.31%**，零散逾期组为 **43.96%**。逾期 3 个月以上时零散样本几乎不存在，因此本数据不足以支持“连续逾期一定更危险”的独立结论。

## 分析方法

1. 获取 UCI 数据并将 `X1…X23` 显式映射为官方业务字段。
2. 检查形状、字段、缺失、重复、标签和还款状态编码。
3. 构造平均账单、平均还款、额度使用率、还款比例、逾期月数和连续逾期指标。
4. 对非正授信额度或非正账单分母返回缺失值，避免产生无穷或误导性比例。
5. 按授信额度、年龄、逾期月份、额度使用率、还款比例、金额四分位和规则风险层级比较违约率。
6. 在总逾期月份相同的条件下比较连续与零散逾期，避免把逾期次数差异误当作连续性效应。

## 数据来源与许可

数据来自 UCI Machine Learning Repository：[Default of Credit Card Clients](https://archive.ics.uci.edu/dataset/350/default+of+credit+card+clients)，DOI：[10.24432/C55S3H](https://doi.org/10.24432/C55S3H)，采用 [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/) 许可。仓库不分发原始数据，运行获取脚本可从官方来源取得。

## 使用限制

数据反映特定地区和历史时期，结论不能直接外推到当前中国大陆信用市场。性别、婚姻状况等字段仅用于描述性检查，不应被简单解释为授信依据。分析只能说明数据中的相关关系，不代表因果关系。本项目仅用于学习和作品展示，不用于真实个人的自动化信贷决策。

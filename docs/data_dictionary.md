# 数据字典

UCI 客户端返回 `X1` 至 `X23`。获取脚本依据官方字段说明将其转换为以下业务名称。

| 字段 | 含义 | 说明 |
|---|---|---|
| LIMIT_BAL | 授信额度 | 包含个人及家庭附属信用额度 |
| SEX | 性别编码 | 仅用于描述性/公平性检查 |
| EDUCATION | 教育程度编码 | 0、5、6 等为未明确类别，分析时不得擅自赋义 |
| MARRIAGE | 婚姻状态编码 | 0 为未明确类别 |
| AGE | 年龄 | 年 |
| PAY_0, PAY_2…PAY_6 | 近六期还款状态 | 正数表示延迟月数；-2、-1、0 属于无逾期状态编码 |
| BILL_AMT1…BILL_AMT6 | 近六期账单金额 | 可为零或负值，可能反映无账单或余额调整 |
| PAY_AMT1…PAY_AMT6 | 近六期还款金额 | 不能在分母为零时机械计算比例 |
| default_next_month | 下一期是否违约 | 1=违约，0=未违约 |

## 派生指标

| 指标 | 定义 | 边界处理 |
|---|---|---|
| avg_bill_amount | 六期账单均值 | 保留负值以便质量检查 |
| avg_payment_amount | 六期还款均值 | 不做主观截断 |
| credit_utilization | 平均账单 / 授信额度 | 授信额度非正时记为缺失 |
| repayment_ratio | 六期还款总额 / 六期正账单总额 | 分母非正时记为缺失 |
| overdue_months | 六期中还款状态大于 0 的月份数 | -2、-1、0 均不算逾期 |
| longest_overdue_streak | 最长连续逾期月数 | 按数据字段的时间顺序计算 |
| risk_segment | 规则式风险层级 | 0 月=低，1–2 月=中，3–6 月=高；不等同真实授信模型 |

> 注意：原始数据存在少量完全重复行。本阶段保留并报告，不在缺少客户唯一标识的情况下武断删除。
